## Install
You can get **`pantry-cloud`** via [npm](https://www.npmjs.com/package/pantry-cloud).
```
npm install pantry-cloud
```

## Usage
```ts
import { Pantry } from 'pantry-cloud';

const pantry = new Pantry('<PANTRY_ID>');
```
