<h1 align="center">pantry-cloud</h1>
<p align="center">
  NodeJS package for <a href="https://getpantry.cloud" target="_blank">Pantry</a>
</p>
<hr>

## Install
```
npm i -S pantry-cloud
```

## Usage
```ts
import { Pantry } from 'pantry-cloud';

const pantry = new Pantry('<ID>');
```
